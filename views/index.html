About Project

This online course management system can register new student in the database, allows them to enroll to the new available courses, as well as give the option to the student to withdraw from the course, The contains 6 tables, (student, teacher, course, courseregistration, teachercourse, semester). The language that we have used to design and developed this project is express.JS for backend basic control passing and for routing, authentication is acheived by using passport.JS
